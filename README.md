# 📘 Student Grade Tracker

A simple Java-based application to manage student records, calculate grades, and generate reports.  
This project was built as part of the **CodeAlpha Internship Program**.

---

## ✨ Features
- Add and manage student details (name, marks).
- Calculate average, highest, and lowest marks.
- Generate a summary report (`students_report.txt`).
- Console-based UI for easy interaction.

---

## 🛠️ Technologies Used
- **Java** (Core Java, OOP concepts)
- **File Handling** (read/write student data)
- **Command Line Interface**
